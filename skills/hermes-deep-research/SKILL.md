Investiga a fondo el siguiente tema: $ARGUMENTS

Pasos obligatorios:
1. Identifica que tipo de investigacion es: proveedor, producto, normativa, HS code, o logistica.
2. Busca en el contexto disponible (MCP tools, web si aplica) toda la informacion relevante.
3. Cruza fuentes: no te quedes con el primer resultado.
4. Si es normativa chilena, verifica con las skills de customs cargadas.
5. Identifica riesgos, costos ocultos y permisos requeridos.

Devuelve un informe conciso con:
- **Hallazgos clave** (3-5 bullets)
- **Evidencia** (con referencias a archivos/URLs/IDs)
- **Recomendacion accionable**
- **Riesgos detectados**

No entregues contenido relleno. Solo lo que el usuario necesita para decidir.
